CSS

CSS, que significa "Cascading Style Sheets" (Folhas de Estilo em Cascata), é uma linguagem que usamos para estilizar páginas da web. É como a "roupa" que vestimos para as nossas páginas, ajudando a deixá-las bonitas e organizadas.
