Scratch

Para praticar programação vamos desfrutar do Scratch.


Scratch é uma linguagem de programação criada em 2007 pelo Media Lab do MIT. Desde 2019 o Scratch 3 está disponível on-line e como uma aplicação para Windows, OS X, e Linux.

Curso em vídeo é uma ótima plataforma para aprender a mexer no Scratch e dominar mais sobre o universo da programação
